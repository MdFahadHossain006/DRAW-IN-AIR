## ✨ About The Project

**DRAW-IN-AIR** is not just a drawing app —  
it transforms your webcam into a futuristic virtual canvas where your hand becomes the brush.

Using **Computer Vision** and **AI Hand Tracking**, users can draw in the air naturally without touching the screen or using any physical device.

This project combines:
- 🧠 Artificial Intelligence
- 👁️ Computer And Mobile Vision
- ✋ Gesture Recognition
- 🎨 Interactive Creativity

into one immersive experience.

# 🧠 How It Works

```text
Webcam → Hand Detection → Finger Tracking → Motion Analysis → Virtual Drawing
```

The system continuously:
1. Captures webcam frames
2. Detects hand landmarks
3. Tracks finger movement
4. Converts movement into drawing strokes
